# Security Whitepaper
Version 3.2
Published: 2026-04-10

## Encryption

Customer data is encrypted in transit using TLS 1.2 or later. Data at rest is encrypted using AES-256.

## Access controls

Northstar Cloud supports role-based access control. Single sign-on using SAML 2.0 is available on Business and Enterprise plans. SCIM provisioning is available only on Enterprise plans.

## Audit logs

Business plans retain audit logs for 90 days. Enterprise plans retain audit logs for 365 days by default. Enterprise customers may purchase extended retention.

## Backups

Production databases are backed up every six hours. Backups are retained for 35 days.

## Compliance

Northstar Cloud maintains SOC 2 Type II certification. The supplied materials do not make any claim about HIPAA compliance.
